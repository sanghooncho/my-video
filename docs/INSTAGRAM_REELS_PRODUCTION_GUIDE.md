# Instagram Reels 제작 가이드 (PreView video-team)

> 목적: Remotion 기반으로 **인스타 릴스용(9:16) 영상**을 만들 때
> - **영상이 화면을 꽉 채우고(full-bleed)**
> - **자막/CTA가 UI에 가려지지 않게(safe zone)**
> - **피드 미리보기(프로필 그리드)에서도 핵심이 보이게**
> 운영 규칙을 고정한다.

## 1) 기본 스펙
- **캔버스:** 1080×1920 (9:16)
- **길이:** 15~90초 범위가 일반적 (플랫폼 정책/가이드 변동 가능)
- **출력:** MP4(H.264) + AAC, 30fps 권장

참고(웹): Sprout Social Reels 가이드에서 Reels가 9:16 비율이며 15~90초 언급.  
Source: https://sproutsocial.com/insights/instagram-reels/

---

## 2) ‘풀스크린’(full-bleed) 규칙
### ✅ 해야 할 것
- 모든 배경 이미지/영상은 **AbsoluteFill로 1080×1920 전체를 덮는다**.
- 원본이 가로/정사각이면:
  - Remotion에서 `objectFit: 'cover'`로 크롭해 꽉 채운다.
- 배경 위 텍스트 가독성 확보:
  - 하단/상단에 **그라데이션 오버레이**(검정 0→40~60%) 또는
  - 블러/글래스모피즘 박스(반투명) 적용

### ❌ 피해야 할 것
- 9:16인데 상하 레터박스(검은 띠) 생기는 구성
- 텍스트가 배경과 대비가 약해서 읽히지 않는 구성

---

## 3) Safe Zone(자막/CTA 안전영역) 규칙
인스타 UI(계정명/설명, 우측 버튼, 하단 캡션/오디오 영역 등)에 의해
텍스트가 가려질 수 있다. 기기/버전마다 차이가 있으므로 **보수적으로** 잡는다.

### 권장 Safe Zone (보수적)
- **좌우 여백:** 최소 80px
- **상단 금지 영역:** 상단 250px (계정명/상태 영역 회피)
- **하단 금지 영역:** 하단 400px (캡션/오디오/버튼 겹침 회피)

즉, 자막/CTA 핵심 텍스트는 대략 아래 영역에 배치:
- X: 80 ~ 1000
- Y: 250 ~ 1520

### 프로필 그리드(미리보기) 대응
프로필 그리드 썸네일은 사실상 **중앙 1:1**로 잘려 보일 수 있음.
따라서 ‘가장 중요한 키워드/얼굴’은 **중앙 1080×1080 안**에 들어오게 설계.

---

## 4) 자막(텍스트) 디자인 규칙
- **한 화면 = 한 문장(또는 2줄 이내)**
- 한글 자막은 Pretendard/Noto Sans KR 계열 권장
- 최소 크기 가이드(1080×1920 기준):
  - 메인 자막: 56~84px
  - 보조 자막: 36~52px
- 줄바꿈은 의미 단위로(조사/접속사 단독 줄바꿈 금지)
- 강조는 색 1개만(브랜드 포인트 컬러)

---

## 5) 영상 편집/연출 체크리스트
- [ ] 첫 1초에 훅(질문/숫자/반전/경고)
- [ ] 2~3초마다 화면 변화(컷/줌/텍스트 전환)
- [ ] BGM/보이스오버 레벨 밸런스
- [ ] CTA는 마지막 1~2초에 명확히(“Link in bio”, “DM”, “무료 체험” 등)
- [ ] Safe zone 밖으로 텍스트가 나가지 않는지 확인

## 6) Remotion 구현 가이드(권장 컴포넌트)
### Safe Area Guide 오버레이 (디버그용)
- 개발 중에만 켜고 렌더에는 끄기.
- 상단 250px / 하단 400px / 좌우 80px 가이드라인 표시.

### Full-bleed Background
```tsx
<AbsoluteFill>
  <Img src={staticFile('bg.png')} style={{width: '100%', height: '100%', objectFit: 'cover'}} />
  {/* overlay */}
  <AbsoluteFill style={{background: 'linear-gradient(180deg, rgba(0,0,0,0.35), rgba(0,0,0,0) 40%, rgba(0,0,0,0.55))'}} />
</AbsoluteFill>
```

## 7) 운영 규칙(이 프로젝트에서 강제)
- 기본 캔버스는 1080×1920(9:16)
- 텍스트/CTA는 Safe Zone 내 배치
- 풀스크린 배경 + 오버레이로 가독성 확보
- 렌더 전 텔레그램으로 검수(영상 파일 전달) 후 업로드
